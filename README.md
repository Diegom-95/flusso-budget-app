# Flusso — Gestione Budget Personale

App Android per la gestione completa del budget mensile e personale.

## Funzionalità

- Registrazione spese e entrate con categorie personalizzabili
- Obiettivi di risparmio con monitoraggio progressivo
- Gestione debiti
- Transazioni ricorrenti automatiche
- Dashboard annuale con grafici
- Export dati in CSV
- Blocco app con PIN

## Stack tecnico

| | |
|---|---|
| **Linguaggio** | Kotlin |
| **Architettura** | MVVM |
| **Database** | Room (SQLite) |
| **UI** | Material Design, Navigation Component, Bottom Navigation |
| **Altro** | ListAdapter, ViewBinding |
